# A11Y GUI

Accessibility testing, with HTML reports

![previewing report](docs/screen1.jpg)

## Installation

    npm install a11y-gui

## Usage

    a11y-gui <url> <html-report-file>
    
    # Example:
    a11y-gui https://www.github.com ./my-report.html
